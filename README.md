# Picture RPA

[中文](#中文) | [English](#english)

---

## 中文

### 简介
基于图像识别的轻量级RPA工具

### 功能
- **内置截图工具**
- **图像识别**
- **逻辑控制**
- **流程编辑**

### 目录结构
- `main.py`: 程序入口
- `gui.py`: UI与事件逻辑
- `screenshot.py`: 截图工具
- `constants.py`: 配置
- `snips/`: 截图保存目录

### 运行
1. **克隆**:
   ```bash
   git clone https://github.com/yiyanQAQ/Picture-RPA-Python.git
   ```
2. **配置环境**:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   pip install -r requirements.txt
   ```
3. **启动程序**:
   ```bash
   python main.py
   ```
