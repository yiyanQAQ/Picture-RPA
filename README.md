# Picture RPA v3.0

[中文](#中文) | [English](#english)

---

## 中文

### 简介
基于图像识别的轻量级RPA工具

### 功能
- **内置截图工具**
- **图像识别**
- **逻辑控制**
- **流程编辑**

### 目录结构
- `pictureRPA.py`: 主程序
- `snips/`: 截图保存目录

### 运行
1. **克隆**:
   ```bash
   git clone https://github.com/your-username/picture_RPA.git
   ```

2. **配置环境**:
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```
3. **启动程序**:
   ```bash
   python pictureRPA.py
   ```
